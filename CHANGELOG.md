# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026

### Features
- 6 themes: Dracula, Monokai, GitHub, Nord, One Dark, Solarized Dark
- Syntax highlighting for 180+ languages with per-theme color maps
- Line numbers (toggle with `--no-line-numbers`)
- Retina support with `--scale 1|2|3` (default: 2x)
- Configurable tab width with `--tab-size`
- Rounded window corners with drop shadow
- macOS-style traffic-light controls
- Auto-sizing canvas based on code content
- PNG export
- Programmatic API via `require('code-to-image')`
